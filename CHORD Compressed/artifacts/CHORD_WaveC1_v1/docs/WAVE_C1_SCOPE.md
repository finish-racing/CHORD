# Wave C1 Scope

Wave C1 implements the PRISM Engine core scoring and reranking layer.

## Included
- candidate generation from canonical tracks
- weighted scoring passes
- score fusion
- confidence scoring
- recommendation explanation generation
- reranking constraints
- safe / balanced / discovery modes
- recommendation persistence tables
- CLI/API surfaces for PRISM analysis output

## Deferred
- standalone curated-playlist builder
- full CLI/operator completion
- full CHORD web interface
- OpenAI integration
- installer and deployment automation
