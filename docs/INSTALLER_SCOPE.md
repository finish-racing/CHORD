# Installer and Upgrade System Scope

This package implements the Ubuntu installer, upgrade, rollback, and reset support for CHORD.

## Included
- install script
- upgrade script
- rollback script
- soft reset script
- hard reset script
- environment file template
- systemd service unit
- Nginx site template
- PostgreSQL bootstrap SQL
- operator install / upgrade / rollback notes
- versioned release directory layout
- stable current symlink policy

## Notes
This package is aimed at the final Ubuntu release path, not per-wave field testing.
