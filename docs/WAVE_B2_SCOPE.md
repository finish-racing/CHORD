# Wave B2 Scope

Wave B2 implements the external enrichment foundation for CHORD.

## Included
- official Last.fm API client
- MusicBrainz API client
- AcousticBrainz API client
- enrichment cache tables
- provenance and confidence storage
- retry-aware HTTP helper
- fallback routing scaffold
- CLI/API inspection surfaces for enrichment

## Deferred
- full duplicate / variant / cover intelligence
- full mood / role / omission intelligence
- PRISM scoring
- full web UI
