# Wave C2 Scope

Wave C2 implements the standalone curated-playlist builder layer for CHORD.

## Included
- curated playlist set persistence
- curated playlist item persistence
- standalone curated-playlist assembly logic
- existing-song pool selection logic
- candidate inclusion logic from PRISM outputs
- diversity / era / role balancing
- omission rescue integration
- boundary protection
- CLI/API surfaces for curated-playlist generation and inspection

## Deferred
- full CLI/operator completion
- full CHORD web interface
- OpenAI integration
- installer and deployment automation
