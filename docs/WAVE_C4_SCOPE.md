# Wave C4 Scope

Wave C4 implements the approved Ubuntu web interface direction for CHORD.

## Included
- FastAPI HTML interface
- upload flow with required Top 25 support
- quiz flow
- run creation flow
- progress thermometer/state display
- analysis dashboard
- recommendation board
- curated-playlist board
- export/download links
- debug/admin visibility when enabled
- CHORD visual system baseline matching the approved UI direction

## Deferred
- OpenAI integration
- installer/deployment automation
